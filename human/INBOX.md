### Как пользоваться
1. Сверху — самое приоритетное. Открой файл из колонки «Файл».
2. Для `DEC`: заполни `answer:` + `status: answered`. Для `HT`: заполни `## Результат` + `status: done`.
3. Эта таблица — индекс; её актуализирует агент-оркестратор. Если расходится с файлами — верь файлам.
4. Счётчики `DEC-####`/`HT-####` — глобальные монотонные; новый номер = max существующий + 1.
