## Назначение
Нативная Rust терминальная сетка для координации параллельных AI-агентских сессий:
мультиоконный ConPTY-менеджмент и оркестрация agent-workflow. Персональная альтернатива
Windows Terminal с кастомной оркестрацией агентов.

## Ответственность / границы
**Входит:** терминальная сетка, ConPTY/PTY, координация сессий.
**НЕ входит:** worktree-изоляция (концептуально — `agent-workspace`).

## Публичный интерфейс / точки входа
Слоистая PoC-архитектура: `poc/` (бинари вех), `crates/` (извлекаемые слои). Текущая веха: PoC 0 (PTY wiring).

## Сборка и тесты
```
cargo run -p poc0-pty
```

## Связи и зависимости
Внешних код-зависимостей от других персональных репо нет. Тематически связан с
`agent-workspace` (оба — про оркестрацию AI-агентов).

## На что смотреть при кросс-репо изменениях
Экспериментальный, рано для жёстких контрактов. Бэкап планов — в `d:\GitHub\Personal\Safe\tessmux\`.
